WHY

為什麼要自己申請 Key？

圖文生成室是「工具」，實際出圖是呼叫 Google 的 Gemini AI。我們讓每位使用者用自己的 Google API Key，這代表：

  • 費用完全透明：出圖費用直接算在你自己的 Google 帳號，我們碰不到、也不從中抽成。
  • 資料更安全：你的 Key 只存在你這台裝置的瀏覽器，不會上傳到我們的伺服器。
  • 額度自己掌控：免費額度、付費升級都由你決定，用多少付多少。
💡 好消息 Google AI Studio 網頁介面與申請 Key 本身永遠免費，台灣可直接連線。多數輕量使用者靠免費額度就夠用。
STEP BY STEP

一、免費申請 API Key

只要有 Google 帳號就能申請，全程約 3 分鐘、不需要信用卡。

1

前往 Google AI Studio

開啟 aistudio.google.com/apikey，用你的 Google 帳號登入。若是第一次使用，會跳出使用條款，勾選同意後繼續。

2

建立 API Key

登入後點畫面上的 Create API key（建立 API 金鑰）。第一次使用時，Google 會自動幫你建立一個預設專案，直接用即可。

3

複製並保存

系統會產生一串 AIza... 開頭的金鑰。點右側複製，貼到你能找回的地方（例如記事本）。這串就是你的 API Key。

4

貼進圖文生成室

回到工具，把這串 Key 貼進去。工具會驗證一次，成功後就記在這台裝置，之後不用再貼。

⚠️ 看到「Free tier」才有免費額度 申請介面若顯示該 Key 的方案是 Free tier 或 Unavailable，代表可用免費額度。若顯示已綁定付費，是因為你的 Google 帳號曾建立過付費專案——這時到 Google Cloud 新建一個「不綁付費」的專案，在該專案底下建立 Key 即可拿回免費額度。

UPGRADE

三、升級付費（額度不夠時）

如果你出圖量大、常遇到免費額度用盡（429 Too Many Requests），可以升級付費層，享有更高額度與更穩定的速度。步驟：

  1. 到 Google AI Studio 的 API Keys 頁面，找到你的 Key，點右側 Set up Billing（設定帳單）。
  2. 依畫面選擇地區、綁定信用卡，建立 Cloud 帳單帳戶。
  3. 2026 年起 Google 採 Prepay（預付） 制：先儲值credits，出圖費用即時從餘額扣抵。餘額歸零時 API 會停止服務（不會無預警爆帳單）。
🚨 付費層沒有硬性花費上限 付費（Postpay）模式下，Google 目前只提供「預算警示」Email 通知，不會在超過額度時自動停止服務。強烈建議：
① 優先用 Prepay 預付制（餘額用完就停，最安全）
② 到 Google Cloud Billing 設定預算警示（例如 $5、$8、$10 時通知你）
💡 新用戶 $300 試用金 Google Cloud 新用戶常有 $300 美元、90 天的試用抵用金。但請注意：自 2026 年 3 月起，部分帳號的 Gemini API 費用可能被排除在此優惠外，實際以你 Google Cloud 帳單頁的顯示為準。
SECURITY

四、安全須知

API Key 是存取你 Google 帳單資源的唯一憑證，等於你帳戶的鑰匙。請務必：

  • 不要외流分享：一旦外流，他人可用你的 Key 大量出圖，帳單算你的。
  • 不要貼在公開場合：不要把 Key 貼到公開的網頁、社群、GitHub、聊天群組。
  • 本工具的做法：你的 Key 只存在你這台裝置的瀏覽器本機，不會上傳、不會存進我們的檔案；工具右上角可隨時一鍵清除。
  • 懷疑外流時：到 Google AI Studio 刪除舊 Key、重新產生一組即可。
⚠️ 2026 金鑰新制 Google 自 2026 年起將 API Key 改為更安全的「Auth keys」，新申請的 Key 都是這種。若你有很舊的 Key 出現 Blocked 標記，重新建立一組新的即可。
FAQ

常見問題

Q：一定要付費嗎？

不一定。免費額度足以試用與輕量使用，只有大量頻繁出圖時才需要升級。

Q：申請要收費或綁卡嗎？

免費層申請 Key 不需要信用卡。只有升級付費層時才需要綁定付款方式。

Q：Key 貼進工具安全嗎？

安全。Key 只存在你這台裝置的瀏覽器，不會上傳到我們或任何第三方伺服器。

Q：換一台電腦要重新貼 Key 嗎？

是的，因為 Key 存在本機。換裝置時重新貼上同一組 Key 即可。
